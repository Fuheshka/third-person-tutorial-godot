# Godot – Third Person Learning Project

This repository contains a small third-person learning project created while continuing to study the Godot Engine. The project follows a tutorial focused on third-person character control and camera behavior.

## Purpose

The purpose of this project is to practice and understand the basics of third-person gameplay implementation in Godot, including character movement in 3D space, camera control, player input handling, animations. The project serves purely as a learning exercise.

## Engine & Language

Godot Engine 4.6  
GDScript

## Project Structure

The project uses a simple and standard Godot structure.

res://  
scenes/ — main scene, player, and camera setup  
scripts/ — character and camera logic  
assets/ — 3D models, materials, and other resources  
project.godot — project configuration file
